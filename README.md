# 電影清單
收藏喜愛電影的網站。
## 網站畫面
![螢幕擷取畫面 2022-05-24 163546](https://user-images.githubusercontent.com/103798145/169988051-cdcd9457-9a2e-4a1e-9791-26dcc377d014.jpg)
## 功能說明
- 從API取得電影資料
- 瀏覽全部電影
- 查看電影詳細資訊
- 使用關鍵字搜尋電影
- 將電影加入/移除我的最愛
- 切換清單與卡片模式
## 環境建置
- Bootstrap 5.1.3
- Fontawesome
- Axios
